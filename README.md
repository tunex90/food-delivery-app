# 🍕 Local Food Delivery App

A food delivery application built for the local community to connect customers with nearby restaurants.

## Features

- 🔍 Restaurant search functionality
- 📍 Geolocation to detect user's current location
- 🏪 Restaurant listings with ratings and delivery info
- 📱 Responsive design for mobile and desktop

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Geolocation API

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Allow location access when prompted

## Project Structure

```
food-delivery-app/
├── index.html      # Main HTML structure
├── style.css       # Styling and layout
├── app.js          # Geolocation functionality
└── README.md       # Documentation
```

## Usage

The app automatically requests your location on load to show nearby restaurants. Browse available restaurants with ratings, estimated delivery times, and fees.
